F1 Driver Performance Under Adverse Conditions: Does Skill or Car Matter More?
Analysis of four seasons (2022–2025) of F1 race results, qualifying data, and real-time session weather to test whether adverse conditions — wet races, street circuits, high altitude — compress the grid and expose genuine driver skill beyond car quality. Built for BUSN 32120 (Data Analysis with Python and SQL, Winter 2026) using the Jolpica F1 API and OpenF1 API.
